# Expense Calculator (Python)

## Project Overview

The Expense Calculator is a beginner-friendly Python project that helps users track their daily or monthly expenses. Users can enter multiple expenses, calculate the total amount spent, and identify the highest expense.

## Features

* Add multiple expenses
* Calculate total expenses
* Find the highest expense
* Simple and interactive command-line interface
* Easy to understand for beginners

## Technologies Used

* Python

## Python Concepts Used

* Variables
* Input/Output
* While Loop
* If Statement
* Arithmetic Operators

## Project Files

* `expense_calculator.py`
* `README.md`
* `Expense_Calculator_Output.png`

## How to Run

1. Open the project in Python or Anaconda.
2. Run `expense_calculator.py`.
3. Enter expense values one by one.
4. Enter `0` to stop entering expenses.
5. View the total expenses and the highest expense.

## Output

The program displays:

* Total Expenses
* Highest Expense
